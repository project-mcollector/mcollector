# MCollector

Data collection and analytics platform designed for scalable event tracking and insights across services

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Starts the web app and API

## Build

```bash
pnpm build
```

## Project Structure

- `apps/web` — Next.js frontend
- `apps/api` — .NET API
- `packages/sdk` — Shared SDK
